Reference Image 技术

剧本分析

输入原始剧本文本，AI 将提取场景、角色、动作与情绪等结构化数据

输入剧本
模型：GPT-4 / Claude
Prompt 模板
System: 你是一个专业的影视剧本分析助手。 User: 请分析以下剧本，提取： - 场景列表（时间、地点、环境） - 角色信息（姓名、外貌、性格） - 分镜建议（镜头类型、景别、时长） - 情绪基调与氛围关键词 [剧本内容]
分析结果将显示在这里

分镜生成

基于剧本分析结果，自动生成分镜脚本，包含镜头类型、景别和时长

生成设置
模型：GPT-4
分镜脚本将显示在这里

图片生成

根据分镜描述和风格指令，使用 DALL-E / Stable Diffusion 生成每个镜头的画面

生成设置
基于分镜脚本批量生成
Prompt 模板
Prompt: Cinematic wide shot, rainy night street, dim streetlights, lonely atmosphere, film grain, 16:9 aspect ratio Negative: cartoon, anime, low quality, blurry, watermark, text, deformed
生成的图片将显示在这里

角色一致性处理

上传角色参考图或生成角色参考，确保同一角色在不同分镜中保持外貌一致

角色参考图

点击上传角色参考图片

支持 JPG / PNG，建议正面清晰半身照

一致性技术
将参考图特征注入所有分镜
技术说明
方案 1：Reference Image • 使用 IP-Adapter 将角色特征编码为向量 • 在每次图像生成时注入该特征向量 • 优点：无需训练，即时生效 方案 2：LoRA 微调 • 使用 5-10 张参考图训练角色 LoRA • 在生成时加载 LoRA 权重 • 优点：一致性更高，支持多角度
一致性处理结果将显示在这里
